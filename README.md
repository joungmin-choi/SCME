# SCME
This is a source code for analyi in "The Impact of Spatial Correlation on Methylation Entropy with Application to Mouse Brain Methylome" paper.

## Requirements
* SRA toolkit (>= 3.0.1)
* TrimGalore (>= 0.6.7)
* Bismark with bowtie2 (>= 2.4.4)
* Samtools (>= 1.4)
